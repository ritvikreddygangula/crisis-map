# Demo Polish Skill

Optimize for judges.

The demo should show:

1. Emergency problem
2. User need
3. Resource network
4. Best recommendation
5. Community report
6. Trust/status update

Keep copy simple.

Strong phrases:

- Find emergency help near you.
- Real-time shelters, water, power, Wi-Fi, and clinics.
- Community reports keep resources trusted and up to date.
- ReliefRoute connects people to the safest available resource.

Before final submission, check:

- Landing page looks clean.
- Demo data is realistic.
- Main flow works.
- README explains setup.
- Pitch is easy to say in 30 seconds.
