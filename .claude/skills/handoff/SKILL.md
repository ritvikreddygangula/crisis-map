# Handoff Skill

Update `HANDOFF.md`.

Include:

1. Current part
2. Completed work
3. Files changed
4. Commands run
5. What works
6. What does not work
7. Blockers
8. Exact next task
9. Recommended next agent
10. Notes for next teammate

The next person should be able to continue without reading the entire chat.

Keep it honest and specific.
