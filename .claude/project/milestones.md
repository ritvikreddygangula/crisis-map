# ReliefRoute Milestones

## Milestone 1: App Shell

Status: Not started

Done when:

- App runs locally.
- Landing page exists.
- Main map page route exists.
- Demo data can be displayed without MongoDB.

## Milestone 2: Database

Status: Not started

Done when:

- MongoDB connects successfully.
- Resource model exists.
- Report model exists.
- Seeded resources appear from API.

## Milestone 3: Map MVP

Status: Not started

Done when:

- Map displays.
- Resource markers appear.
- Clicking a marker shows details.
- Filters work.

## Milestone 4: Reporting

Status: Not started

Done when:

- User can submit a report.
- Report is saved.
- Resource status/trust score updates.
- Updated info appears in UI.

## Milestone 5: Demo Ready

Status: Not started

Done when:

- Main demo flow works.
- UI is clean.
- README is clear.
- Pitch script is ready.
- No obvious build errors.
